# Vuepress 搭建教程

  1. 第一步，初始化项目

npm init

  1. 第二步，安装 vuepress

npm install -D vuepress

  1. 创建你的第一篇文档

mkdir docs && echo '# Hello VuePress' > docs/README.md

  1. 在 package.json 中添加一些 scripts
  "scripts": {
    "dev": "vuepress dev docs --temp .temp",
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs",
  },
